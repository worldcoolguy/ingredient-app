# Ingredients App

A web application that allows users to manage ingredients. This is built by Vue.js, Vuex and Vuetify.

Techs:

* Vue.js
* Vuex
* Vuetify
* Scss

## Getting Started

```bash
# Install the dependencies.
$ cd .
$ npm install
```

### Run the App

```bash
$ cd .
$ npm run serve
```

By default, you can visit the app in your web browser at `http://localhost:8080`
